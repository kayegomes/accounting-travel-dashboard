## 🔄 Processar Nova Planilha

### Passo a Passo

1. **Substitua o arquivo**:
   ```bash
   cp sua_planilha.xlsx /home/ubuntu/upload/PainelContábilV2-copia.xlsx
   ```

2. **Execute novamente**:
   ```bash
   python3.11 executar_analise_final.py
   ```

3. **Pronto!** Novos arquivos gerados.
